# Gráficos avanzados y visualización de datos

Técnicas de visualización que comunican información de forma efectiva y profesional.

## Gráficos combinados

Combinan dos tipos de gráfico en uno:

1. Selecciona los datos
2. Insertar → **Gráfico combinado**
3. Asigna cada serie a un tipo (columna, línea) y eje

**Ejemplo**: Barras para ventas + línea para objetivo en el mismo gráfico.

## Gráfico de cascada (Waterfall)

Muestra cómo un valor inicial se modifica por incrementos positivos y negativos:

1. Insertar → Cascada
2. Ideal para: análisis de ganancias/pérdidas, variaciones presupuestarias

```
Ingresos: +50,000
Costos: -30,000
Impuestos: -5,000
= Ganancia neta: 15,000
```

## Gráfico de embudo

Visualiza procesos con etapas decrecientes:

- Leads: 1000
- Contactados: 600
- Interesados: 300
- Propuestas: 150
- Cerrados: 75

## Gráfico de mapa

Visualiza datos geográficos:

1. Datos con nombres de países/estados
2. Insertar → Mapa

## Minigráficos avanzados (Sparklines)

Personalización de sparklines:

1. Insertar → Sparklines (línea, columna, ganancia/pérdida)
2. En **Diseño de sparkline**:
   - Resaltar punto alto/bajo
   - Primer/último punto
   - Puntos negativos
   - Color y peso de línea

## Técnicas de diseño profesional

### Eliminar ruido visual

- Quita líneas de cuadrícula innecesarias
- Reduce bordes y elementos decorativos
- Usa colores con propósito, no decorativos

### Jerarquía visual

- Resalta la serie más importante con un color intenso
- Usa gris para datos de contexto
- El título debe contar la historia, no solo describir

### Etiquetar directamente

En lugar de leyenda separada, coloca etiquetas junto a las líneas/barras.

## Gráficos dinámicos (Pivot Charts)

Gráficos vinculados a tablas dinámicas:

1. Selecciona la tabla dinámica
2. Insertar → Gráfico dinámico
3. El gráfico se actualiza al filtrar/modificar la tabla

Incluyen botones de filtro integrados.

## Dashboards con gráficos

### Layout profesional

1. Crea los gráficos en hojas separadas
2. En la hoja dashboard, copia los gráficos como imágenes vinculadas:
   - Copiar gráfico → Pegado especial → **Imagen vinculada**
3. Organiza en una cuadrícula limpia

### Formularios de control

Usa controles de formulario como filtros:
- **Lista desplegable**: Desarrollador → Insertar → Cuadro combinado
- **Botones de opción**: Para alternar entre vistas
- **Barra de desplazamiento**: Para parámetros numéricos

Vincula los controles a celdas que alimenten los gráficos con fórmulas.

## Resumen

Los gráficos avanzados y las técnicas de visualización profesional transforman datos en historias comprensibles. La clave es simplicidad, propósito y consistencia visual.
