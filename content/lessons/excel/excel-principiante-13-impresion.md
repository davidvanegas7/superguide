# Impresión y configuración de página

Preparar una hoja para impresión profesional es una habilidad esencial.

## Vista previa de impresión

`Ctrl + P` o **Archivo** → **Imprimir**

Aquí ves exactamente cómo se verá tu documento impreso.

## Configuración de página

Pestaña **Diseño de página** → **Configurar página**:

### Orientación

| Opción | Ideal para |
|--------|-----------|
| **Vertical** | Datos con pocas columnas |
| **Horizontal** | Datos con muchas columnas |

### Márgenes

| Tipo | Superior/Inferior | Izquierdo/Derecho |
|------|-------------------|-------------------|
| Normal | 1.91 cm | 1.78 cm |
| Estrecho | 1.27 cm | 0.64 cm |
| Ancho | 2.54 cm | 2.54 cm |

### Tamaño de papel

- Carta (Letter): 21.59 × 27.94 cm
- A4: 21 × 29.7 cm
- Oficio (Legal): 21.59 × 35.56 cm

## Área de impresión

Para imprimir solo una parte de la hoja:

1. Selecciona el rango
2. Pestaña **Diseño de página** → **Área de impresión** → **Establecer área de impresión**

Para eliminarla: **Borrar área de impresión**

## Ajustar escala

Si tus datos no caben en una página:

- **Ajustar a 1 página de ancho por X de alto**
- O ajustar el porcentaje de escala manualmente (10% - 400%)

## Imprimir títulos

Para repetir encabezados en cada página:

1. Diseño de página → **Imprimir títulos**
2. **Repetir filas en extremo superior**: selecciona la fila de encabezados
3. **Repetir columnas a la izquierda**: si necesitas

## Encabezado y pie de página

Pestaña **Insertar** → **Encabezado y pie de página**

Elementos automáticos disponibles:
- Número de página
- Número total de páginas
- Fecha y hora
- Nombre del archivo y hoja
- Ruta del archivo

**Ejemplo**: `Página &[Página] de &[Páginas]` → "Página 1 de 5"

## Saltos de página

### Ver saltos de página

**Vista** → **Vista previa de salto de página**

Las líneas azules muestran dónde se dividirá la impresión. Puedes arrastrarlas para ajustar.

### Insertar salto manual

1. Selecciona la fila donde quieres el corte
2. Diseño de página → **Saltos** → **Insertar salto de página**

## Resumen

Configurar la página correctamente ahorra papel y tiempo. Define orientación, márgenes, área de impresión y encabezados repetidos antes de imprimir para resultados profesionales.
