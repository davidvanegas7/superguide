# Atajos de teclado esenciales

Los atajos de teclado multiplican tu productividad. Aquí están los imprescindibles.

## Navegación

| Atajo | Acción |
|-------|--------|
| `Ctrl + Inicio` | Ir a A1 |
| `Ctrl + Fin` | Ir a la última celda con datos |
| `Ctrl + →/←/↑/↓` | Saltar al límite del rango |
| `Ctrl + G` | Ir a una celda específica |
| `Ctrl + AvPág/RePág` | Cambiar de hoja |
| `F5` | Cuadro de diálogo Ir a |

## Selección

| Atajo | Acción |
|-------|--------|
| `Ctrl + A` | Seleccionar todo |
| `Ctrl + Shift + Fin` | Seleccionar hasta la última celda |
| `Ctrl + Barra espaciadora` | Seleccionar columna completa |
| `Shift + Barra espaciadora` | Seleccionar fila completa |
| `Ctrl + Shift + →` | Seleccionar hasta el final del rango |

## Edición

| Atajo | Acción |
|-------|--------|
| `F2` | Editar celda activa |
| `Escape` | Cancelar edición |
| `Enter` | Confirmar y bajar |
| `Tab` | Confirmar y mover a la derecha |
| `Ctrl + Z` | Deshacer |
| `Ctrl + Y` | Rehacer |
| `Ctrl + C` | Copiar |
| `Ctrl + X` | Cortar |
| `Ctrl + V` | Pegar |
| `Supr` | Borrar contenido |
| `Ctrl + -` | Eliminar celdas/filas/columnas |
| `Ctrl + +` | Insertar celdas/filas/columnas |

## Formato

| Atajo | Acción |
|-------|--------|
| `Ctrl + 1` | Formato de celdas |
| `Ctrl + B` | Negrita |
| `Ctrl + I` o `Ctrl + K` | Cursiva |
| `Ctrl + U` o `Ctrl + S` | Subrayado |
| `Alt + Enter` | Salto de línea dentro de la celda |

## Fórmulas

| Atajo | Acción |
|-------|--------|
| `=` | Iniciar fórmula |
| `Alt + =` | Autosuma |
| `F4` | Alternar tipo de referencia ($) |
| `Ctrl + Shift + Enter` | Fórmula matricial (Legacy) |
| `F9` | Evaluar parte de la fórmula |
| `Ctrl + `` ` | Mostrar/ocultar fórmulas |

## Datos y archivo

| Atajo | Acción |
|-------|--------|
| `Ctrl + S` | Guardar |
| `Ctrl + P` | Imprimir |
| `Ctrl + N` | Nuevo libro |
| `Ctrl + W` | Cerrar libro |
| `Ctrl + Shift + L` | Activar/desactivar filtros |
| `Alt + D + S` | Ordenar |

## Trucos avanzados

### Rellenar hacia abajo
Selecciona el rango (incluyendo la celda con dato) → `Ctrl + D`

### Rellenar hacia la derecha
Selecciona el rango → `Ctrl + R`

### Introducir el mismo dato en varias celdas
1. Selecciona todas las celdas
2. Escribe el valor
3. `Ctrl + Enter` (no solo Enter)

### Seleccionar celdas especiales
`Ctrl + G` → **Especial** → elige: fórmulas, constantes, celdas en blanco, etc.

## Resumen

No necesitas memorizar todos los atajos de una vez. Empieza con los de navegación y edición, luego incorpora los de formato y fórmulas. Con la práctica, se vuelven automáticos.
